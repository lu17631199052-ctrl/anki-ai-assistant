使用指南

各功能详细说明

快速掌握每个功能的最佳用法。

🧠 左侧工具栏

安装后 Anki 窗口左侧自动出现固定图标栏。图标从上到下依次为：记事本、待办清单、AI 对话、AI 错题整理、AI 生成卡片、浏览器搜索、设置。点击图标在右侧展开对应面板，再次点击或按关闭按钮收起。可通过菜单栏「AI Assistant → 📋 左侧工具栏」重新打开。

💬 AI 对话

打开后自动获取当前复习卡片的正反面内容作为上下文。输入问题后 AI 流式回复，完整支持 Markdown 语法。每个回复下方有操作按钮：复制全部内容、写入卡片背面、创建新卡片。点击右上角按钮在侧边栏固定和浮动窗口间切换。AI 回复期间可继续输入新问题，消息自动排队。

🔍 AI 解释卡片

在复习界面选中任意卡片，按快捷键触发。AI 自动读取卡片正面（题目）和背面（答案），流式输出深度解析。解析窗口底部可一键将内容添加到卡片背面或创建新卡片。

📝 AI 生成卡片

支持多种输入方式：粘贴文本、上传文件（txt / md / pdf / docx / 图片）、粘贴图片。扫描版 PDF 自动调用视觉模型 OCR 识别。上传后可自定义生成数量、目标语言、目标牌组，预览确认后一键导入。

❌ AI 错题整理

截取错题图片或粘贴到窗口 → 视觉模型自动识别题目内容和选项 → AI 解析正确答案并生成选择题（MCQ）→ 预览确认后添加到专用"选择题"笔记类型。复习时正面点击选项交互，背面自动显示对错。

🌐 浏览器搜索

选中卡片上的文字后打开搜索面板，自动带入选中文字。顶部可切换搜索引擎（百度 / Google / Bing / B 站 / YouTube），也可点击「在外部浏览器打开」跳转到系统默认浏览器。

常见问题

FAQ

使用中遇到问题？先看看这里。更多问题欢迎到 GitHub 提 Issue。

手机版 Anki（AnkiMobile / AnkiDroid）能用吗？
不能。本插件基于 PyQt 开发，是桌面版 Anki 专用插件。AnkiMobile（iPhone/iPad）和 AnkiDroid（Android）不支持 Qt 插件系统，无法安装任何 Python 插件。
插件安装后看不到左侧工具栏？
重启 Anki 后，左侧工具栏会在主窗口加载完成后自动出现（约 0.5 秒延迟）。如果仍看不到，检查 Anki 版本是否 ≥ 23.10。也可以通过菜单栏「AI Assistant → 📋 左侧工具栏」手动打开。
如何获取 API Key？
前往对应平台的开发者页面注册并创建 API Key。大部分平台注册即送免费额度，足够日常使用。具体地址见上方安装配置板块。
快捷键不生效怎么办？
Mac 用户请确认使用的是 Control (^) 键而非 Command (⌘)。Windows 用户需同时按 Ctrl+Shift。如果仍有问题，可通过菜单栏「AI Assistant」使用鼠标操作。
可以用本地模型吗？
可以！选择 Ollama 提供商，在本地安装 Ollama 并拉取模型（如 qwen2.5:7b），填入本地地址即可完全离线使用。视觉功能也可配置本地多模态模型。
支持哪些视觉模型？
支持所有主流视觉模型：DeepSeek-Vision（deepseek-v4-flash 自动支持图片）、通义千问 VL（qwen-vl-plus / qwen-vl-max）、智谱 GLM-4V、GPT-4V 等。视觉模型可在设置中独立配置。
生成的卡片格式是什么？
默认保留原始 Markdown 格式存储到 Anki 字段。如果在设置中开启"Markdown 转 HTML"，则转换为 HTML 存储。选择题（MCQ）使用专属"选择题"笔记类型，由模板内嵌 JS 渲染交互效果。
